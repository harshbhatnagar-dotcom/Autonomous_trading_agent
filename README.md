# 📈 Autonomous Trading Agent

An AI-powered autonomous trading agent built using OpenAI Agents SDK, MCP Servers, Groq LLMs, Tavily Search, Polygon Market Data, and Gradio.

## Features

- Autonomous portfolio management
- Financial news research
- Stock market analysis
- MCP-based tool integration
- Account and strategy management
- Gradio dashboard
- Playwright support
- Docker & Hugging Face Spaces compatible
